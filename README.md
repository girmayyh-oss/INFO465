# INFO 465 Final Tableau Dashboard

## Project Overview

This repository contains evidence of our final Business Intelligence solution for INFO 465. The project focuses on helping DPS and VADOC better understand procurement spending, vendor activity, and SWAM participation through Tableau dashboards.

Our goal was to turn raw procurement data into clear visuals that support faster reporting, better decision-making, and improved compliance tracking.

Most of our work was initially developed and tracked in Google Docs, and later organized into this repository to demonstrate our use of GitHub and Git for version control and collaboration.

This project was completed by Team Yellow:  
Bethlehem Habte, Breanna Obeng, Faustina Koduah, Yanit Girmay, and Yabby Wondwosen.

## Dashboard Features

The final Tableau dashboard includes:

- Total Spend by Fiscal Year and Fiscal Quarter
- SWAM Participation Rate by Fiscal Year
- Vendor Spending Trend
- Filters for Fiscal Year, Fiscal Quarter, and Vendor Name
- Forecast indicator for vendor spending trends
- Tableau Story/Journey mode leading into the final dashboard
